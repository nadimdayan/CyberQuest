# CyberQuest 🛡️

A Cybersecurity Awareness Quiz Web Application built 
to educate users about online safety and cyber threats.

## 🌐 About The Project
CyberQuest is an interactive quiz platform focused on 
cybersecurity awareness. Users can test their knowledge 
on topics like phishing, passwords, malware, and more.

## 🛠️ Technologies Used
- PHP
- HTML5
- CSS3
- JavaScript
- MySQL Database

## ⚙️ How to Run This Project
1. Install XAMPP on your computer
2. Start Apache and MySQL in XAMPP Control Panel
3. Clone or download this repository
4. Paste the project folder into C:\xampp\htdocs\
5. Import the database .sql file into phpMyAdmin
6. Open your browser and go to localhost/CyberQuest

## 📂 Database Setup
1. Open phpMyAdmin → localhost/phpmyadmin
2. Create a new database named cyberquest
3. Click Import → choose the .sql file
4. Click Go
